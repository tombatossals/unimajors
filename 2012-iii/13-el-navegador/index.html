El navegador web

Un navegador o navegador web (del inglés, web browser) es un programa que permite ver la información que contiene una página web (ya se encuentre ésta alojada en un servidor dentro de la World Wide Web o en un servidor local).

Los navegadores Web

Hoy veremos algunas de las características principales de un navegador web, que podemos resumir en lo siguiente:

  • Nos permite conectarnos a la web.
  • Aplicación MUY SENCILLA de manejar. Debe ser de los programas más accesibles para un usuario.
  • Muchos navegadores a elegir. Chrome, Firefox, Explorer son los principales.
  • La web es algo cambiante, necesita de navegadores modernos.

¿Qué navegador vamos a utilizar?

Es indiferente, ya que los tres tienes una funcionalidad similar. Además que es conveniente aprender a manejar más de uno. En cualquier caso, os doy mi recomendación:

  • Google Chrome. Es el navegador más moderno y avanzado hoy en día. Y el que yo os recomiendo.
  • Mozilla Firefox. Navegador muy moderno y actual y ampliamente soportado. Muy recomendable.
  • Internet Explorer. Navegador de Microsoft.

Decir que Internet Explorer lo tendréis instalado en cualquier ordenador Windows, mientras que los otros dos navegadores deben instalarse previamente.

Repaso direcciones web

Hagamos un recordatorio de lo que hemos visto en las dos sesiones anteriores al respecto de las direcciones web.

  • Las direcciones web son la manera de referenciar un documento en la web.
  • No llevan acentos, ni espacios, ni caracteres especiales.
  • Las mayúsculas y minúsculas no afectan al dominio, pero sí a la ruta del documento.

Ejemplos direcciones web

Veamos algunos ejemplos de direcciones web.

  • http://mayores.uji.es
  • http://google.es
  • http://flickr.com
  • http://whitehouse.gov
  • http://es.wikipedia.org

La dirección web (URL)

La URL (Uniform Resource Locator) es la manera que tenemos de acceder a los contenidos de la web.

  • Como si de un callejero se tratase, para navegar necesitamos conocer direcciones.
  • Páginas web hay miles de millones. Cada una de ellas tiene una dirección web que la identifica.
  • No hace falta aprenderse todas. Tenemos varias ayudas en la navegación para poder acceder a todos los contenidos sin tener que memorizar cada URL.

La navegación con buscadores

Hace unos años encontrar contenidos en la web era complicado, ya que necesitábamos conocer uno o varios enlaces de los que partir para encontrar la información que buscábamos. Hoy en día es más fácil gracias a los motores de búsqueda.

  • El buscador es una página web. Sólo deberemos recordar la URL del buscador para tener el punto de partida.
  • Conoce e indexa toda la información. Cuando le preguntemos por un tema nos dirá cuáles son las páginas más importantes relacionadas.

Resumen

  • La navegación requiere práctica. Tenemos que practicar mucho para dominar la técnica de la navegación.
  • Tenemos ayudas. Los navegadores y buscadores día a día nos ponen las cosas más sencillas para encontrar y acceder a los recursos que necesitamos.
  • Merece la pena el esfuerzo. Los contenidos que podemos encontrar en la web superan ampliamente las expectativas que tengáis, os lo aseguro.
