El ordenador

Máquina electrónica que recibe y procesa datos para convertirlos en información útil. Formada principalmente por los siguientes componentes:

  • Torre.
  • Ratón.
  • Teclado.
  • Pantalla.

El arranque del ordenador

En informática, la secuencia de arranque, (boot o booting en inglés) es el proceso que inicia el sistema operativo cuando el usuario enciende una computadora. Se encarga de la inicialización del sistema y de los dispositivos.

¿Qué es el sistema operativo?

El Sistema operativo (SO) es el software que actúa de interfaz entre los dispositivos de hardware y los programas usados por el usuario para utilizar un computador.

Los botones de acción de ventana

Los botones de acción de ventana los veremos en cualquier aplicación que abramos en Windows 7.

  • Maximizar Al pulsarlo la ventana tomará el 100% del espacio disponible en el escritorio.
  • Cerrar Al pulsarlo la aplicación se cerrará definitivamente.
  • Minimizar Al pulsarlo la aplicación se ocultará.
  • Restaurar Si una ventana está maximizada, al pulsarlo la aplicación volverá a su tamaño original.

El ratón (o mouse)

  • Fundamental para la navegación. Sin ratón se haría muy complicado navegar por Internet.
  • Desplazamiento del puntero por la pantalla. Al mover el ratón reconoce el movimiento mediante tecnología óptica/láser.
  • Tiene 2 botones principales. El botón izquierdo realiza la acción principal, el derecho la acción secundaria.
  • Ruedecilla central. Opcional, muy útil para desplazar páginas.
  • Drag&Drop. Técnica fundamental.

¿Qué debemos saber de un teclado?

  • Utiliza una disposición de teclas similar a la antigua máquina de escribir..
  • Combinaciones de teclas. Muchos carácteres o comandos especiales se aplican mediante combinaciones de teclas.

Aplicaciones

En informática, una aplicación es un tipo de programa informático diseñado como herramienta para permitir a un usuario realizar uno o diversos tipos de trabajos.

Ejemplos de aplicaciones:

  • Navegador. Chrome, Firefox, Internet Explorer.
  • Editor de textos. WordPad, Word 2010...
  • Retoque fotográfico.
  • Juegos.

WordPad

Wordpad es una aplicación de Edición de textos básica que encontraremos en todos los sistemas operativos Windows.

Principales secciones que podemos encontrar en Wordpad:

  • Zona central de edición. Es la hoja en blanco donde podemos escribir nuestro texto utilizando el teclado y el cursor de edición.
  • Barra de herramientas con iconos Todas las utilidades con las que podremos formatear nuestro documento están en la zona superior y utilizaremos el ratón para hacer uso de ellas.

Cortar y pegar

Es una técnica básica en la edición de textos que nos ayuda a tratar con información en bloque.

  • Se selecciona el texto que queremos mover/duplicar. Mediante el ratón, tal y como vimos la clase anterior.
  • Se elige la opción deseada desde la barra de herramientas. Copiar o pegar.
  • Se posiciona el cursor de edición donde queramos situar el texto. Mediante el ratón, llevamos el cursor a la posición deseada.
  • Pulsamos el botón de pegar de la barra de herramientas.
