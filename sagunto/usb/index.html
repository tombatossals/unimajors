Dispositivos externos

Hoy en día es muy común conectar dispositivos al equipo mediante un cable USB plug&play

  • Todo tipo de dispositivos: Cámaras de fotos, impresoras, Webcams, tarjetas de sonido, móviles, etc.
  • Tipos de conectores usb: Estándar, micro-USB, USB-3.0, etc.
  • Windows 7/XP gestiona de manera automática estos dispositivos. El extremo del PC siempre es igual, el extremo del dispositivo puede ser diferente.

Gestión del dispositivo USB

El extremo del PC siempre es un USB standard, y al conectarlo se iniciará el proceso de autodetección por parte de Windows 7/XP.

  • Conexión por primera vez. W7 deberá detectar el dispositivo y encontrar el driver adecuado para comunicarse con él.
  • Una vez detectado. Si todo ha ido bien, el dispositivo se registrará en el sistema y podremos acceder a él directamente.
  • Las posteriores ocasiones que lo conectemos la detección será mucho más rápida.

Compresión de archivos

La compresión de datos es la reducción del volumen de datos tratables para representar una determinada información empleando una menor cantidad de espacio. Al acto de compresión de datos se denomina compresión, y al contrario descompresión

Tipos de compresiones:

  • Sin pérdidas. La información comprimida es exactamente igual a la información descomprimida.
  • Con pérdidas. La información comprimida pierde información en la compresión.

Documentos comprimibles

Dependiendo del tipo de información a comprimir, se utilizarán unos algoritmos (programas) u otros para conseguirlo.

Tipos de documentos comprimibles:

  • Documentos de datos. Muy comprimibles, sin pérdidas. DOC, XLS, TXT, HTML, PDF, RTF.
  • Imágenes/Video. Muy comprimibles, con pérdida. JPG, PNG, AVC, DIVX, etc.
  • Audio Muy comprimibles, con pérdida. MP3.

Algoritmos sin pérdidas

Utilizan técnicas de búsqueda de redundancias, y los podemos utilizar con las aplicaciones Winzip, Winrar, 7-zip o similares.

  • Muy útiles para archivos de texto

Algoritmos con pérdidas

Utilizan técnicas de eliminación lo que el ojo/oido humano no puede percibir, y se aplican con software específico de cada tipo de archivos.

  • Muy útiles para archivos multimedia

JPG: Compresión de imágenes con perdidas

Reduce el tamaño de la imagen aprovechando que el ojo humano no es capaz de captar determinados cambios de brillo y color en zonas con amplios contrastes.

  • RAW, TIF, BMP. Formatos de imagen sin compresión.
  • GIF, PNG. Formatos de imagen con compresión sin pérdidas.
  • JPG. Formato de imagen que utiliza compresión con pérdidas.

    • Para transformar una imagen de un formato a otro utilizamos un programa de tratamiento fotográfico como Photoshop, Gimp o Paint.NET.

MP3: Compresión de audio con perdidas

Reduce el tamaño de archivo al eliminar todo el rango dinámico del sonido que el sistema auditivo humano no puede diferenciar.

  • WAV. Formato de audio sin compresión.
  • FLAC. Formatos de compresión sin pérdida.
  • MP3, WMA, AAC, OGG. Formatos de compresión con pérdida.

Winzip

Programa que nos va a permitir gestionar aquellos documentos que nos envíen comprimidos.

  • .ZIP. Extensión de archivos comprimidos.
  • Archivos embebidos. Cada archivo comprimido puede tener varios archivos.
  • Extraer. Es el proceso de recuperar un documento de un archivo comprimido.

Winrar

Otro programa de gestión de archivos comprimidos.

  • .ZIP o .RAR. Puede gestionar este tipo de archivos comprimidos, entre otros.
  • Permite crear archivos .RAR muy fácilmente.
  • Útil para enviar por correo varios archivos.
