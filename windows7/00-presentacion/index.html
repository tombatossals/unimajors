Objetivos del curso

  • Conocer y coger práctica con Windows 7.
  • Identificar las principales diferencias con Windows XP.
  • Perder el miedo al cambio de sistema operativo.

Metodología empleada

  • Un ordenador por persona.
  • Curso empírico. El conocimiento lo alcanzaremos a través de la práctica.
  • Nivel medio. Presupondremos ciertos conocimientos previos con Windows XP.
  • Clases dinámicas. Preguntad lo que queráis en cualquier momento.

Objetivos del curso

Saber cómo trabajar con un equipo con Windows 7 de igual manera que lo hacíamos con un equipo con Windows XP.

Objetivos del curso

Identificar las partes propias de un sistema operativo, y comprobar que realmente no debe afectar para nada en nuestra manera de trabajar el que nos lo cambien.

Objetivos del curso

Perder el miedo, o no preocuparnos, cuando mañana nos digan que ya no hay Windows 7 que ahora hay Windows 8. :)
