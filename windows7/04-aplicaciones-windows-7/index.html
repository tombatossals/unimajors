Aplicaciones

En informática, una aplicación es un tipo de programa informático diseñado como herramienta para permitir a un usuario realizar uno o diversos tipos de trabajos.

Ejemplos de aplicaciones:

  • Navegador. Chrome, Firefox, Internet Explorer.
  • Editor de textos. WordPad, Word 2010...
  • Retoque fotográfico.
  • Juegos.

Aplicaciones en Windows 7

Hay algunas aplicaciones que ya vienen pre-instaladas en Windows 7, y que podemos utilizar directamente.

Algunas de las aplicaciones preinstaladas en Windows 7:

  • Internet Explorer. El navegador de Microsoft.
  • WordPad. Editor de textos.
  • Calculadora. Calculadora muy simple.
  • Solitario. Juego de cartas.
  • Paint. Aplicación de dibujo.

  • Chrome y Firefox. Navegadores, gratuitas.
  • Photoshop. Retoque fotográfico, de pago.

Aplicaciones de pago, aplicaciones gratis

El mundo del software rige su economía siguiendo un sistema de licencias con dos vertientes muy diferenciadas:

  • Licencias privativas. El software es desarrollado por una empresa, que cobra un dinero por cada instalación que se hace de su aplicación.
  • Licencias abiertas. El software es libre y gratuito para el que quiera instalárselo, en todos los equipos que desee.

Piratería

Se llama piratería al hecho de instalar una aplicación sin haber cumplido los términos estipulados en su correspondiente licencia.

Si el software que nos acostumbramos a utilizar es software con licencias abiertas, NUNCA incurriremos en un delito de piratería.

Instalar nuevas aplicaciones

Para poder instalar nuevas aplicaciones necesitamos una cuenta con privilegios de Administrador.

En los PC's de la UJI no disponemos de ese tipo de cuenta, por lo que no podemos instalar aplicaciones de la manera habitual que debería hacerse.

En vuestros ordenadores de casa SÍ que podréis instalar software como ahora veremos. Además, os comentaré otra opción que podemos utilizar para instalar software sin privilegios de administrador.

PortableApps

Es un tipo especial de instalación donde no requeriremos permisos de administrador, ya que todas las aplicaciones se instalan dentro de una carpeta.

  • Podemos llevar todas las aplicaciones en un llavero USB.
  • No todas las aplicaciones pueden ser portables.
  • Muy útil para descubrir nuevas aplicaciones.
